# 虚拟桌面切换特效
<!--more-->
[KDE](https://github.com/zzag/kwin-effects-cube)
[GNOME](https://github.com/Schneegans/Desktop-Cube)

好像GNOME的可玩性更高一点(仅指这个扩展)，改天试试Debian+GNOME。
### Install for KDE and ...
```sh
yay -S kwin-effects-cube
```
安装完后依次点击 `工作区行为 -> 桌面特效 -> Cube` 进行设置
